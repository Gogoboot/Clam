## 🗓 Рекомендуемый порядок

```
1.  Вынести утилиты в utils/               — быстро, чистота кода
2.  Таймаут для LLM fetch                  — быстро, надёжность
3.  Дебаунс saveSettings                   — быстро, производительность
4.  Валидация URL LLM                      — быстро, надёжность
5.  Переключатель языка Whisper            — заметная польза
6.  Drag & Drop                            — заметное UX улучшение
7.  useMemo + useCallback                  — производительность
8.  Разделение на хуки                     — упростит дальнейшую разработку
9.  Разделение на компоненты               — после хуков станет легче
10. Горячие клавиши                        — быстрая победа
11. Ползунок яркости                       — UI улучшение
12. Скачивание моделей                     — убирает ручную установку
13. Кэширование транскрипций               — экономит время
14. Мультипровайдер LLM                    — расширяет аудиторию
15. Очередь файлов                         — сложно, но очень полезно
16. История транскрипций                   — долгосрочная ценность
17. Встроенный плеер                       — завершает основной функционал
18. Диаризация                             — продвинутая фича
```
